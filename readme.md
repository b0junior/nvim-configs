## Poisonline:

-   Work in progress
-   Requisitos :
    `nvim 0.5+`
    `lua`
    `luaJIT`

O arquivo `poisonline.lua` deve ser movido para `~/.config/nvim/lua` e
carregado no arquivo `init.vim`.
Ex:

    lua require('poisonline')
